# 谷歌搜索居中 / Google Search Center

将谷歌搜索结果页面布局居中显示，改善宽屏下的阅读体验。

## 功能

- 当屏幕宽度大于 1537px 时，自动将搜索结果居中显示
- 通过 CSS 和 inline style 双重保证居中效果
- 监听 DOM 变化，兼容 [AutoPager](https://greasyfork.org/zh-CN/scripts/419215-autopager) 等自动翻页脚本
- 轻量无侵入，不影响页面原有功能

## 效果对比

### 使用前

搜索结果默认靠左，宽屏下右侧大量留白。

### 使用后

搜索结果居中显示，阅读更舒适。
